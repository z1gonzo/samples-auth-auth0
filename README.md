## Usage

```sh
# CREATE TODO
POST /todos
# GET ALL TODOS
GET /todos
# GET A TODO
GET /todos/:id
# UPDATE A TODO
PUT /todos/:id
# DELETE A TODO
DELETE /todos/:id
```
